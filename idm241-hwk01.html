RULES

Like - Default

  • Hover ON - Cursor changes to finger
  • Hover OFF - Cursor changes to pointer
  • 1 second delay "I like this" displays in the tagline
  • Mouse Down - Button color changes to a darker shade
  • Mouse Up - Icon outline fills in and plays an animation
  • Database record flag set to TRUE

Like - Active

  • Hover ON - Cursor changes to finger
  • Hover OFF - Cursor changes to pointer
  • 1 second delay "Unlike" displays in the tagline
  • Mouse Down - Button color changes to a darker shade
  • Mouse Up - Icon filled in icon changes back to outline
  • Database record flag set to FALSE
