RULES

Tile - Default

Hover On

  • Cursor changes to finger
  • Game tile shifts up 10px
  • Game tile has a drop shadow
  • Game title changes from dark grey to red

Hover Off

  • Cursor changes to pointer
  • Game tile shifts down 10px
  • Game tile drop shadow is removed
  • Game Title changes from red to dark grey

  • Game tile color changes to a darker shade

Mouse Up

  • Navigates to the Nintendo's information game page
