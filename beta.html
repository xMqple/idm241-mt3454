Beta Description

www.nintendo.com

TRIGGERS

Tile - Default

  • Hover ON
  • Hover OFF
  • On Click (Mouse Up)

RULES

Hover On

  • Cursor changes to finger
  • The game card shifts up 5px with a smooth transition.
  • The game card has a shadow behind.
  • Game Title changes from dark grey to red

Hover Off

  • Cursor changes to pointer
  • Game image shifts downward 10 pixels
  • Game Title changes from red to dark grey

On Click

  • Navigates to the Nintendo's information game page
