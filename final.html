RULES

Tile - Default

Hover On

  • Cursor changes to finger
  • Game tile shifts up 10px
  • Game tile has a drop shadow
  • Game title changes from dark grey to red

Hover Off

  • Cursor changes to pointer
  • Game tile shifts down 10px
  • Game tile drop shadow is removed
  • Game Title changes from red to dark grey

Mouse Down

  • Game tile color changes to a darker shade

Mouse Up

  • Navigates to the Nintendo's information game page

Like Button - Default

Hover On

  • Icon fills in with dark grey

Hover Off

  • Icon filled in icon changes back to outline

Mouse Up

  • Icon fills in with red
  • Database record flag set to TRUE

Like Button - Active

Mouse Up

  • Icon filled in icon changes back to outline and changes to dark grey
  • Database record flag set to FALSE

Quanity - Minus

Hover On

  • Icon color changes from dark grey to red.

Hover Off

  • Icon color changes from red to dark grey.

Mouse Up

  • Quanity number subtracts by 1
  • If quanity number hits 0, number won't subtract by 1.

Quanity - Plus

Hover On

  • Icon color changes from dark grey to red.

Hover Off

  • Icon color changes from red to dark grey.

Mouse Up

  • Quanity number adds by 1
  • If quanity number hits 99, number won't add by 1.

Add to Cart - Default

Hover On

  • Button changes from dark grey to red.

Hover Off

  • Button changes from red to dark grey

Mouse Up

  • Text changes from "Add To Cart" to "Add to Cart" and a checkmark icon.
  • Game image translates to into the shopping cart.
  • Cart value icon shows indicated number from quanity button.

Add to Cart - Active

Hover On

  • Button changes from red to dark grey.

Hover Off

  • Button changes from dark grey to red.

Mouse Up

  • Text changes from "Add To Cart" and a checkmark icon to "Add to Cart".
  • Cart value icon is removed.
